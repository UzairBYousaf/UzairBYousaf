E-commerce website built using Django + Python

This is Amazon/Flipkart/Daraz clone like website built by using Python programming language with Django as a framework.

I built this website as a fun project to learn about Django, Microservices & how E-commerce works as a whole.

The website houses both frontend as well as backend to ensure full functionalities. It also makes use of Google Maps API so that users can custom pick a delivery location.

The website does not require any sign-in or registeration of any account, the user can simply make use of their payment cards to complete an order.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - HTML/CSS
  • - JavaScript ES6
  • - Google Maps API
  • - XML2PDF - PDF generator
