GoTravel - Travel Agency Website

This is a self learning project I created, as a part of me learning web development using MERN Stack. This website comes with both frontend and backend programming, and uses Node.js, React.js, Redux, React Hooks, GraphQL & MongoDB.

The website consists of a homepage, from where you can fill a form to create a new tour, or you can select one of the pre-destined tours availale on the tour page.

The web app consists of both, an admin page and an client page, as an admin you can add new tours, change existing ones or delete them at will. As a client you can mange your selected tours, or you can create a new tour or manage your newly created tour.

The projects also consists of a payment page on which you can select various methods of payment and proceed with your tour.

Technologies:
  • - HTML/CSS
  • - JavaScript ES6
  • - React JS
  • - Node JS
  • - Redux
  • - React Hooks
  • - GraphQL
  • - MongoDB
